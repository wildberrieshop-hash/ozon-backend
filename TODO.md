## 🔄 Текущая версия (v0.1 Beta)

Текущая реализация поддерживает:
- Базовый поток создания поставки
- Проверку доступности товара
- Интеграцию с Озон API

### Известные ограничения:

1. **Кросс-докинг** — требует ввода warehouse_id вручную
   - Текущая версия не позволяет выбрать конкретный кросс-докинг склад в боте
   - Нужно добавить интерфейс для выбора

2. **Данные водителя** — не реализовано
   - При кросс-докинге нужна информация о водителе и машине
   - Текущая версия пропускает эти поля

3. **История поставок** — не видна в боте
   - Поставки сохраняются в БД, но нет команды для просмотра истории
   - Нужна команда `/history` или `/my_supplies`

4. **Отслеживание статуса** — базовое
   - Только создание поставки
   - Нет проверки статуса после создания

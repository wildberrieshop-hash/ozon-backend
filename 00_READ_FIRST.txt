                         ⚡ БЫСТРЫЙ СТАРТ ⚡
================================================================================

1. УСТАНОВИ ЗАВИСИМОСТИ:
   cd TGBOT
   pip install -r requirements.txt

2. СОЗДАЙ .env ФАЙЛ:
   copy .env.example .env

3. ДОБАВЬ TELEGRAM TOKEN:
   Открой .env в блокноте
   Добавь: TELEGRAM_BOT_TOKEN=твой_токен_от_BotFather

4. ЗАПУСТИ БОТ:
   python main.py

5. ИСПОЛЬЗУЙ В TELEGRAM:
   /start - приветствие
   /setup - вводишь Client-ID и API-Key
   /create - создаешь поставку

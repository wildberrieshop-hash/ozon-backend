## ⚡ Быстрый старт (3 минуты)

### Шаг 1: Установи зависимости
```bash
cd TGBOT
pip install -r requirements.txt
```

### Шаг 2: Создай .env файл
```bash
# Windows
copy .env.example .env

# Linux/Mac
cp .env.example .env
```

### Шаг 3: Добавь Telegram Bot Token
Открой `.env` в блокноте и добавь токен от @BotFather:
```
TELEGRAM_BOT_TOKEN=123456:ABC...
```

### Шаг 4: Запусти бот
```bash
python main.py
```

Видишь `🚀 Бот запущен!` ? ✅ Готово!

### Шаг 5: Используй в Telegram
- Найди своего бота по username
- `/start` — приветствие
- `/setup` — вводишь Client-ID и API-Key
- `/create` — начинаешь создавать поставку

## 🔐 Безопасность

✅ **Твои данные в безопасности:**
- Client-ID и API-Key хранятся в локальной БД
- Ничего не отправляется на удаленные серверы
- Каждый пользователь имеет свои учетные данные

⚠️ **Помни:**
- Не делись своим API-Key с никем
- Не коммитьте `.env` файл в git
- Компьютер, где запущен бот, должен быть в безопасности

## ❓ FAQ

**В: Нужен ли интернет?**
А: Да, для работы с Озон API. Компьютер должен быть подключен к интернету.

**В: Где хранятся мои учетные данные?**
А: В локальной БД `warehouse.db` (в родительской папке). Только на твоем компьютере.

**В: Могу ли я использовать это коммерчески?**
А: Конечно! Это твой инструмент.

**В: Что если я забыл свой Client-ID?**
А: Откой личный кабинет Озона → Настройки → API.

**В: Как отменить поставку через бот?**
А: Текущая версия не поддерживает. Отменяй через личный кабинет Озона.

**В: Может ли работать несколько пользователей?**
А: Да! Каждый вводит свои учетные данные через `/setup`.

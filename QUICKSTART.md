# ⚡ Быстрый старт (5 минут)

## 1️⃣ Получи Telegram Bot Token (3 мин)

- Открой Telegram → найди `@BotFather`
- `/start` → `/newbot`
- Выбери имя и username (должен заканчиваться на `_bot`)
- Скопируй полученный токен

## 2️⃣ Установи зависимости (1 мин)

```bash
cd TGBOT
pip install -r requirements.txt
```

## 3️⃣ Создай .env файл (30 сек)

Скопируй содержимое из `.env.example` в новый файл `.env`:

```
TELEGRAM_BOT_TOKEN=твой_токен_от_BotFather
```

## 4️⃣ Запусти бот (30 сек)

```bash
python main.py
```

Если видишь `🚀 Бот запущен!` — всё работает! ✅

## 5️⃣ Используй в Telegram (1 мин)

- Открой своего бота в Telegram
- `/start` — приветствие
- `/setup` — вводишь Client-ID и API-Key от Озона
- `/create` — начинаешь создавать поставку

## 🔐 Где хранятся данные?

- **Учетные данные Ozон** → локальная БД `warehouse.db` (в родительской папке)
- **История поставок** → там же
- **Telegram Bot Token** → `.env` файл (локально)

Ничего не отправляется на удаленные серверы! ✅

## ❓ Где найти Client-ID и API-Key?

1. Открой https://seller.ozon.ru
2. **Настройки → API**
3. Там будут твой **Client-ID** и **API-Key**

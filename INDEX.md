## 📂 ВСЕ ФАЙЛЫ ПРОЕКТА

### 🔴 ИСПОЛНЯЕМЫЙ КОД (нужны для запуска)

#### `main.py` (18 KB)
- **Что:** Основной Telegram-бот
- **Содержит:**
  - Класс `OzonSupplyBot` с регистрацией обработчиков
  - FSM диалоги (`SupplyForm`, `SetupForm`)
  - Команды: `/start`, `/setup`, `/create`
  - Обработчики кнопок и текстового ввода
- **Когда трогать:** Нужно добавить новую команду или изменить логику диалога
- **Строк кода:** ~650

#### `supply_flow.py` (9.6 KB)
- **Что:** Полный поток создания поставки
- **Содержит:**
  - Поиск товара по SKU
  - Проверка остатков
  - Создание черновика поставки
  - Скоринг (проверка доступности)
  - Получение таймслотов
  - Создание финальной поставки
- **Когда трогать:** Нужно изменить логику проверок или добавить новые шаги
- **Строк кода:** ~280

#### `tgbot_db.py` (2.8 KB)
- **Что:** Работа с БД пользователей Telegram
- **Содержит:**
  - Функция инициализации таблицы `tg_users`
  - Сохранение учетных данных
  - Получение учетных данных
  - Проверка регистрации пользователя
- **Когда трогать:** Нужно добавить новые поля в профиль или изменить схему
- **Строк кода:** ~90

#### `requirements.txt` (53 B)
- **Что:** Список зависимостей проекта
- **Содержит:**
  - `aiogram==3.4.1` — фреймворк для Telegram ботов
  - `python-dotenv==1.0.0` — загрузка .env конфигов
  - `requests==2.31.0` — HTTP клиент (для API)
- **Использование:** `pip install -r requirements.txt`

#### `.env.example` (291 B)
- **Что:** Пример конфигурационного файла
- **Содержит:** Переменные окружения (безопасно коммитить)
- **Как использовать:** Скопируй в `.env` и добавь свой токен

#### `.env` (не в репо)
- **Что:** Твой конфигурационный файл
- **Содержит:** `TELEGRAM_BOT_TOKEN=...` (чувствительный, в .gitignore)
- **Кто создает:** Ты сам из .env.example

## 🚀 БЫСТРЫЙ СТАРТ (СКОПИРУЙ-ВСТАВЬ)

```bash
# 1. Перейди в папку
cd TGBOT

# 2. Установи зависимости
pip install -r requirements.txt

# 3. Создай .env файл (Windows)
copy .env.example .env
# или (Linux/Mac)
cp .env.example .env

# 4. Открой .env в блокноте и добавь токен:
# TELEGRAM_BOT_TOKEN=123456:ABC...

# 5. Запусти бот
python main.py

# 6. Открой Telegram и найди своего бота
# Напиши /start
```

# Пошаговая инструкция по запуску Telegram-бота

## Требования
- Python 3.10+
- Интернет соединение
- Аккаунт в Telegram
- Client-ID и API-Key от личного кабинета Озона

## Шаг 1: Установка зависимостей

Открой командную строку (cmd) в папке `TGBOT` и выполни:

```bash
pip install -r requirements.txt
```

Если команда не работает, попробуй:
```bash
python -m pip install -r requirements.txt
```

## Шаг 2: Получение Telegram Bot Token

1. Открой Telegram и найди бота `@BotFather`
2. Напиши `/start`
3. Выполни команду `/newbot`
4. BotFather попросит ввести имя бота (например: "OzonSupplyBot")
5. Затем попросит ввести username для бота (должен заканчиваться на `_bot`, например: `ozon_supply_test_bot`)
6. BotFather вернет токен вида:
   ```
   123456789:ABCdefGHIjklmnoPQRstuvWXYZ1234567890
   ```

## Шаг 3: Создание файла .env

1. В папке `TGBOT` найди файл `.env.example`
2. Скопируй его и переименуй в `.env` (или откройте в блокноте и сохраните как .env)
3. Открой `.env` в текстовом редакторе
4. Вставь полученный токен:
   ```
   TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklmnoPQRstuvWXYZ1234567890
   ```
5. Сохрани файл

## Шаг 4: Запуск бота

В командной строке выполни:

```bash
python main.py
```

Если всё работает, ты увидишь:
```
🤖 Инициализирую БД...
🚀 Бот запущен!
```

## Шаг 5: Использование бота в Telegram

1. Открой Telegram
2. Найди своего бота по username (например: `@ozon_supply_test_bot`)
3. Напиши `/start`
4. Следуй инструкциям бота

### Первоначальная настройка:
- Выполни `/setup`
- Введи свой **Client-ID** от личного кабинета Озона
- Введи свой **API-Key** от личного кабинета Озона

### Создание поставки:
- Выполни `/create`
- Следуй шагам:
  1. Введи SKU товара
  2. Введи количество
  3. Выбери схему доставки
  4. Выбери кластеры (галочки)
  5. Выбери дату поставки
  6. Подтверди создание

## Где найти Client-ID и API-Key?

1. Открой личный кабинет Озона (https://seller.ozon.ru)
2. Перейди в **Настройки → API**
3. Там будет твой **Client-ID** (цифры, например: `123456`)
4. И **API-Key** (строка вида: `abc123def456...`)
5. Скопируй их и используй при выполнении `/setup` в боте

## Решение проблем

### Ошибка: "ModuleNotFoundError: No module named 'aiogram'"
**Решение:** Убедись, что установил зависимости:
```bash
pip install -r requirements.txt
```

### Ошибка: "TELEGRAM_BOT_TOKEN not found in .env"
**Решение:** Проверь, что файл `.env` находится в папке `TGBOT` и содержит:
```
TELEGRAM_BOT_TOKEN=твой_токен_здесь
```

### Бот не отвечает на команды
**Решение:** Попробуй перезапустить бот (Ctrl+C и заново `python main.py`)

### Ошибка при вводе Client-ID или API-Key
**Решение:** Убедись, что они скопированы полностью и без пробелов. Их можно найти в настройках личного кабинета Озона.

## Остановка бота

Нажми `Ctrl+C` в командной строке, где запущен бот.

## Далее

После успешного запуска:
- Бот будет работать локально на твоем компьютере
- Убедись, что компьютер включен и подключен к интернету для работы бота
- Тестируй функциональность создания поставок
- Все данные (учетные данные, история поставок) хранятся в локальной базе данных `warehouse.db`
